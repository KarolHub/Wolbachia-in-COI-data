Scripts and data used for comparing COI primer matches across sequences of Wolbachia and other Bacteria in the article: "Insect COI barcoding data as an untapped resource for surveying Wolbachia symbioses".

consensus.py - generates consensus sequence for individual Wolbachia supergroups based on the sequences provided;
metaconsensus.py - generates a global consensus based on all Wolbachia sequences, treating each supergroup as equal weight;
get_match_percent.py - calculates the overall match percentages between the primer sequences and provided Wolbachia references;
